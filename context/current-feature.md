### Incident-fix checklist (hard acceptance criteria ‚Äî port verbatim, assert in pytest)
1. force-unlock by GCS object **generation**, never the JSON `ID` UUID (`common.sh`, `posix/lock-contention.sh`).
2. `tofu output -json`, never `-raw` ‚Äî `-raw` prints #26991 box to stdout, exits 0, poisons downstream (`run.sh`).
3. `down` uses ZERO `-exclude` ‚Äî 2+ flags silently no-op the whole plan (OpenTofu 1.12.3); state-rm the two `prevent_destroy` secrets, destroy, re-import (`suspend.sh`).
4. dump-verify BEFORE any destroy ‚Äî export ‚Üí verify non-empty ‚Üí delete-empty ‚Üí retry once (`posix/dump.sh`, `db.sh`).
5. restore skips when instance was already live ‚Äî never clobber newer data (`db.sh`, `suspend.sh:was_already_live`).
6. reconcile adopt-vs-destroy ‚Äî 5 branches; `AUTO_APPROVE` self-heals via adopt, never destroys unattended (`reconcile.sh`).
7. `-refresh=false` fallback only on the vanished-resource signature, never blanket (`run.sh`).
8. PSC-detach retry is operator-confirmed, never silent auto-retry (`suspend.sh`).
9. IAM propagation cooldown (120s) + provisioning marker spans the whole apply (`run.sh`).
10. `require_kube_context` glob guard before every kubectl-mutating step.
11. kubeconfig-safe overlap ordering in resume + fail-fast join.
12. AR-writable `testIamPermissions` poll (works under WIF where `gcloud config get-value account` is empty).
13. interrupt-safe abort ‚Äî let in-flight `tofu apply` finish persisting state on SIGINT (Python: install SIGINT handler that forwards to the subprocess and `wait()`s, never `os._exit` mid-write).
14. newest ENABLED secret version, never `access latest` (`posix/secrets.sh`).

### Library choices
- **stdlib `subprocess`** ‚Äî replaces every `$(...)`/pipe; `Result{stdout,stderr,code}` + `ProcError`. `long_running()` (Popen + signal) for the interrupt-safe tofu abort (#13). **No third-party runner ‚Äî `sh` is banned (user, 2026-07-08); spiked and rejected** (Unix-only, floor-incompatible, and it *terminates* the child on SIGINT so fix #13 would need re-implementing on top of it). `proc.py` is the single stdlib exec path.
- **tenacity** ‚Äî adopted for CLI-zone retries (`state_lock` network-retry, AR poll), replacing hand-rolled `while` loops. Floor-banned (`shared/dump.py` retry stays stdlib); in the import-grep guard.
- **pydantic v2** ‚Äî parse `tflock`, `tofu output -json`, `gcloud --format=json` (CLI only); stdlib `json` + dataclasses in the Cloud Build path.
- **native urllib/httpx (CLI)** ‚Äî Spaceship DNS API + AR `testIamPermissions` + Monitoring/GitHub API. Cloud Build path stays stdlib `urllib`.
- **typer** ‚Äî subcommand dispatch (`gcp`/`local`/`ci` sub-apps), honoring `AUTO_APPROVE`.
- **structlog** (CLI) / stdlib `logging`+JSON (Cloud Build) ‚Äî structured logs, run-id correlation, secret redaction.

### Cutover strategy (big-bang, staged internally, reversible)
Build whole Python layer + full pytest on a branch ‚Üí prove parity ‚Üí cut over in one commit deleting the shell layer. Port bottom-up: `shared/proc` ‚Üí shared core ‚Üí common/tofu/state_lock/ar ‚Üí bootstrap/db/dns/gke ‚Üí reconcile ‚Üí suspend ‚Üí run.sh dispatch ‚Üí ci/* ‚Üí local ‚Üí Cloud Build last. **Parity gate:** every bats assertion has a pytest equivalent; argv-parity harness diffs old-bash vs new-Python emitted invocations (byte-identical); read-only live checks (`status`/`logs`/`verify-secrets`); one real `suspend`‚Üí`resume` cycle on dev + a no-op `apply` confirming reconcile self-disables; observability + rollback rehearsal. **Rollback:** tag `infra-shell-final` + revert; `_USE_LEGACY_SUSPEND=1` fallback for the first release.

### Risks
- **Biggest:** silently dropping an incident fix. Mitigation = the 14-item checklist, each ‚Üí pytest assertion + rationale comment + live suspend‚Üíresume gate.
- **Effort:** large. reconcile + suspend are the bulk ‚Äî port branch-for-branch with their bats suites in lockstep, do not "improve".
- **Floor import discipline (single 3.14 floor):** a typer/pydantic/structlog import leaking into `shared/` (or anything beyond the pinned `vendored` set into `cloudbuild/`) breaks Cloud Build silently until a suspend fires; guarded by import discipline + the floor-drift guard + the runtime floor assertion. (The version split is gone ‚Äî the auto-suspend path runs gcloud's bundled 3.14, not the image's system 3.13.)
- **New PyPI supply-chain surface** (operator CLI only): mitigated by uv `--frozen`+hashes+`UV_MALWARE_CHECK`+`exclude-newer`; auto-suspend path fetches nothing.
